1 What is Microsoft IQ?

Microsoft IQ is a set of capabilities that form the enterprise intelligence layer of the Microsoft stack. Together its components give AI agents the context they need to reason accurately about your business — your data, your work, your policies, and the web.

Why a "layer"?Agents are only as good as the context they can ground on. Microsoft IQ is the shared layer that supplies that context to Copilot, Copilot Studio agents, and Azure AI Foundry agents alike.

2 The four IQs

One complete viewEach IQ covers a different slice of context. Combined, they let an agent answer a question that spans your data, your collaboration, your policies, and the world.

3 Why it matters

Grounded answers

Agents reason over trustworthy, organization-specific context instead of guessing.

Permission-aware

Context respects the identity and existing access of the user or agent.

Reusable

The same intelligence layer serves many agents across many surfaces.

4 How agents use Microsoft IQ

Agents tap Microsoft IQ through standard protocols and tooling:

  • In Copilot Studio, add IQ capabilities as knowledge/context for an agent.
  • Via the Model Context Protocol (MCP) and agent APIs for pro-code agents.
  • Through Microsoft 365 Copilot, which is grounded in Work IQ and can reach the other IQs.
PreviewParts of Microsoft IQ are in preview and evolving. Avoid hard-coded dependencies on preview tool names and parameters.
