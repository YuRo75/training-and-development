1 What is Foundry IQ?

Foundry IQ is one of three IQ workloads in Microsoft IQ that give agents access to different aspects of your organization. Part of Azure AI Foundry, it provides agents with knowledge — your organization's policies and authoritative documents.

The "what's true" pillarIf Work IQ knows how you work and Fabric IQ knows your data, Foundry IQ knows your authoritative knowledge — the documents and policies an agent should treat as the source of truth.

2 What it provides

Authoritative documents

The canonical content an agent should ground answers in.

Organizational policies

Rules and guidance the agent must respect.

Retrieval

Relevant knowledge fetched and supplied to the model at run time.
Grounding done rightFoundry IQ handles retrieval-augmented grounding so your Foundry agents answer from approved knowledge, not the model's training data alone.

4 Add it to an agent

  1. In Azure AI Foundry, connect your authoritative knowledge sources.
  2. Attach a Foundry IQ knowledge base to a hosted agent.
  3. The agent retrieves relevant knowledge at run time and cites it.
  4. Combine with Work IQ / Fabric IQ for a complete organizational view.
Code or low-codeBuild agents in the Foundry portal, or with the Foundry SDK (Python, C#, TypeScript, Java) and REST — see the quickstarts in the learning path.
