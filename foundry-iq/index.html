1 What is Foundry IQ?

Foundry IQ is one of three IQ workloads in Microsoft IQ that give agents access to different aspects of your organization. Foundry IQ, part of Azure AI Foundry, provides agents with knowledge — your organization's policies and authoritative documents.

The "what's true" pillarIf Work IQ knows how you work and Fabric IQ knows your data, Foundry IQ knows your authoritative knowledge — the documents and policies an agent should treat as the source of truth.

2 What it provides

Authoritative documents

The canonical content an agent should ground answers in.

Organizational policies

Rules and guidance the agent must respect.

Retrieval

Relevant knowledge fetched and supplied to the model at run time.
Grounding done rightFoundry IQ handles the retrieval-augmented grounding so your Foundry agents answer from approved knowledge rather than the model's training data alone.

3 Relationship to Fabric IQ & Work IQ

Microsoft provides three IQ workloads that give agents access to different aspects of your organization:

WorkloadLayerGrounds agents in
Fabric IQSemantic intelligence for FabricBusiness data — ontologies, models, graphs over OneLake & Power BI
Work IQContextual intelligence for M365Collaboration signals — docs, meetings, chats, workflows
Foundry IQKnowledge for Foundry agentsAuthoritative documents and policies

4 How to use it

  1. In Azure AI Foundry, connect your authoritative knowledge sources.
  2. Attach Foundry IQ knowledge to an agent so it grounds responses on that content.
  3. The agent retrieves relevant knowledge at run time and cites it.
  4. Combine with Work IQ / Fabric IQ for a complete organizational view.
Preview & evolvingFoundry IQ is part of the evolving Microsoft IQ family; capabilities and APIs may change. Avoid hard-coded dependencies on preview interfaces.
